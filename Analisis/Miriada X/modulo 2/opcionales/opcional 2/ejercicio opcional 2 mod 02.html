Fuentes Vinciladas

En este ejercicio quiero mostrar como vincular una fuente exterma de una pagina web

  1. Ir a la página web de google fonts
  2. Seleccionar la fuente que más te gusta. En este caso es Roboto.
  3. Abrir la pestaña inferior y copiar el link que esta en STANDARD y pegarlo en la etiqueta < Head >.
    Ejemplo:
    <head>
    <meta charset="utf-8">
    <title>Fuentes vinculadas</title>
    <link href="https://fonts.googleapis.com/css?family=Roboto" rel="stylesheet">
    </head>
  4. Después debes copiar el texto que está en Specify in CSS que en este caso es : font-family: 'Roboto', sans-serif; y pegarlo en la etiqueta de < Style > dentro de body, p o h1.
    Ejemplo:
    <style type="text/css">
    body {font-family: 'Roboto', sans-serif;}
    </style>

Si sigues estos pasos podrás vincular la fuente externa sin problemas.
